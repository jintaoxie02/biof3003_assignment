Building a Context-Aware Chatbot for PPG Health Monitoring

1.Setting Up OpenRouter and Getting a Free API Key

Before integrating the chatbot, you need to set up an account with OpenRouter and obtain a free API key.

Steps:

  1. Sign Up for OpenRouter :

    • Visit the OpenRouter website and create an account.

    • Log in to your dashboard.

  2. Generate an API Key :

    • Navigate to the "API Keys" section in your OpenRouter dashboard.

    • Generate a new API key by clicking the "Create New Key" button.

    • Copy and securely store this key (e.g., in a .env.local file).

  3. Browse Free Models :

    • OpenRouter provides access to various free models like deepseek/deepseek-r1-distill-llama-70b:free.

    • Browse the available models and select one that suits your needs.

Step 1.1 Download and Install Ollama

  1. Visit the official website: https://ollama.ai

  2. Download the installer for your OS (Windows/macOS/Linux)

  3. Run the installer and ensure the Ollama service starts automatically

2. Create a Simple Chatbot

Now that you have your API key, letâ€™s create a basic chatbot by setting up the backend API route, frontend chatbot component, and integrating it into the app.

Step 1: Set Up the Backend API Route

The backend API route (app/api/chatbot/route.ts) handles requests to the OpenRouter API.

Code :

// app/api/chatbot/route.ts
import { NextResponse } from 'next/server';

const OPENROUTER_API_URL = "https://openrouter.ai/api/v1/chat/completions";
const OPENROUTER_API_KEY = process.env.OPENROUTER_API_KEY;

export async function POST(request: Request) {
  try {
    const { message } = await request.json();

    if (!message) {
      return NextResponse.json(
        { success: false, error: 'Message is required' },
        { status: 400 }
      );
    }

    const response = await fetch(OPENROUTER_API_URL, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${OPENROUTER_API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        model: "deepseek/deepseek-chat:free",
        messages: [
          { role: "system", content: "You are a helpful assistant." },
          { role: "user", content: message }
        ],
        max_tokens: 50,
        temperature: 0.7
      })
    });

    const data = await response.json();
    if (!response.ok) {
      throw new Error(data.error.message || 'Error calling OpenRouter API');
    }

    const botReply = data.choices[0].message.content;
    return NextResponse.json(
      { success: true, reply: botReply },
      { status: 200 }
    );
  } catch (error: any) {
    console.error('Error calling OpenRouter API:', error.message);
    return NextResponse.json(
      { success: false, error: error.message },
      { status: 500 }
    );
  }
}

Code Explanation :

  • The POST method receives user messages and sends them to the OpenRouter API.

  • The response from OpenRouter is returned to the frontend.

Step 2: Create the Chatbot Component

The Chatbot component (app/components/Chatbot.tsx) manages the chat interface and sends user messages to the backend API.

Code :

// app/components/Chatbot.tsx
import { useState } from 'react';

interface ChatMessage {
  sender: 'user' | 'bot';
  text: string;
}

export default function Chatbot() {
  const [input, setInput] = useState('');
  const [messages, setMessages] = useState<ChatMessage[]>([]);
  const [loading, setLoading] = useState(false);

  const sendMessage = async () => {
    if (!input.trim()) return;

    const newMessage: ChatMessage = { sender: 'user', text: input };
    setMessages((prev) => [...prev, newMessage]);
    setInput('');

    setLoading(true);

    try {
      const response = await fetch('/api/chatbot', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ message: input })
      });

      const data = await response.json();
      if (data.success) {
        const botMessage: ChatMessage = { sender: 'bot', text: data.reply };
        setMessages((prev) => [...prev, botMessage]);
      } else {
        console.error('Error:', data.error);
      }
    } catch (error) {
      console.error('Error sending message:', error);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div>
      <div>
        {messages.map((msg, index) => (
          <div key={index}>
            <strong>{msg.sender === 'user' ? 'You' : 'Bot'}:</strong> {msg.text}
          </div>
        ))}
      </div>
      <input
        type="text"
        value={input}
        onChange={(e) => setInput(e.target.value)}
        placeholder="Type a message..."
      />
      <button onClick={sendMessage} disabled={loading}>
        {loading ? 'Sending...' : 'Send'}
      </button>
    </div>
  );
}

Code Explanation :

  • The sendMessage function sends user input to the backend API and displays the AI's response.

  • Messages are stored in the messages state to display the conversation history.

3. Add Advanced Features

To make the chatbot more interactive, modify the system prompt to provide context or role-play as an assistant.

Step 1: Modify the System Prompt

Update the system prompt in app/api/chatbot/route.ts to include instructions for the AI.

// app/api/chatbot/route.ts
import { NextResponse } from 'next/server';

const OPENROUTER_API_URL = "https://openrouter.ai/api/v1/chat/completions";
const OPENROUTER_API_KEY = process.env.OPENROUTER_API_KEY;

export async function POST(request: Request) {
  try {
    const { message, avgHeartRate, avgHRV } = await request.json();

    if (!message) {
      return NextResponse.json(
        { success: false, error: 'Message and subjectId are required' },
        { status: 400 }
      );
    }

    // Construct a personalized greeting or context for the chatbot
    let systemPrompt = `
You are HeartLens, an AI assistant for a health monitoring app called "HeartLens." 
Your role is to assist users with their heart rate and HRV data and provide guidance on using the app.

Instructions for Using the App:
- To measure your heart rate and HRV, place your finger gently over the camera lens of your device.
- Ensure the camera is stable and the lighting is adequate for accurate readings.
- Start recording by clicking the "Start Recording" button. The app will analyze your PPG signal using the camera feed.
- Once the recording is complete, save your data to MongoDB for future reference.

If the user asks how to use the app, explain the above steps clearly and concisely.
If the user asks about their heart rate or HRV, use the provided historical data to give personalized responses.
`;
    if (avgHeartRate && avgHRV) {
      systemPrompt += `
        The user's average heart rate is ${avgHeartRate} BPM, and their average HRV is ${avgHRV} ms.
        Use this information to provide personalized responses.
      `;
    }

    const response = await fetch(OPENROUTER_API_URL, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${OPENROUTER_API_KEY}`,
        'HTTP-Referer': 'https://your-site-url.com',
        'X-Title': 'Your Site Name',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        model: "deepseek/deepseek-r1-distill-llama-70b:free",
        messages: [
          { role: "system", content: systemPrompt },
          { role: "user", content: message }
        ],
        max_tokens: 1000,
        temperature: 0.7
      })
    });

    const data = await response.json();
    if (!response.ok) {
      throw new Error(data.error.message || 'Error calling OpenRouter API');
    }

    const botReply = data.choices[0].message.content;
    return NextResponse.json(
      { success: true, reply: botReply },
      { status: 200 }
    );
  } catch (error: any) {
    console.error('Error calling OpenRouter API:', error.message);
    return NextResponse.json(
      { success: false, error: error.message },
      { status: 500 }
    );
  }
}
