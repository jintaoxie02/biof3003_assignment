1. Install Tailwind CSS

Explanation

  1. Container Styling :

    • p-6: Adds padding around the main container.

    • max-w-4xl: Limits the maximum width of the container for better readability.

    • mx-auto: Centers the container horizontally.

    • space-y-6: Adds vertical spacing between child elements.

  2. Title Styling :

    • text-2xl: Sets the font size to large.

    • font-bold: Makes the title bold.

    • text-center: Centers the title horizontally.

  3. Component Containers :

    • Each child component is wrapped in a div with bg-gray-100, p-4, rounded-lg, and shadow-md classes:

      • bg-gray-100: Adds a light gray background.

      • p-4: Adds padding inside the container.

      • rounded-lg: Rounds the corners of the container.

      • shadow-md: Adds a subtle shadow for depth.

  4. Button Styling :

    • px-4 py-2: Adds horizontal and vertical padding to the button.

    • bg-blue-500: Sets the background color to blue.

    • text-white: Sets the text color to white.

    • rounded: Rounds the corners of the button.

    • hover:bg-blue-600: Changes the background color on hover.

    • transition duration-300: Adds a smooth transition effect.
